# MF Analyser Frontend

React + Redux Toolkit + Material UI frontend for the MF Analyser backend.

## Setup

```bash
npm install
npm run dev
```

## Configuration

- Backend base URL is set in `.env` as `VITE_API_BASE_URL` (default `http://localhost:3000`).

## Features

- Sector rankings and top funds (1Y/3Y/5Y)
- Fund lookup with details + sector ranking
